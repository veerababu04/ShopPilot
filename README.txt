ShopPilot - E-commerce Website (Frontend)
==========================================

Hey! This is a simple e-commerce website frontend called ShopPilot. It's just
plain HTML, CSS and images, nothing fancy going on behind the scenes yet.
Good for practicing frontend stuff or as a starting point for a bigger project.

What's inside
-------------
- index.html         -> the homepage, shows off the products
- login.html         -> login page for members
- signup_page.html   -> registration/signup page for new users
- about.html         -> about us page
- contact_us.html    -> contact page
- success.html       -> shows up after someone registers successfully
- images/            -> all the product photos, icons, logo, payment icons etc.

How to run it
--------------
No installation needed, it's just plain HTML
1. Unzip the folder if you haven't already.
2. Just double-click index.html and it'll open in your browser.
3. Or if you want to use a local server (better for testing), you can use
   the Live Server extension in VS Code, or run:
       python3 -m http.server
   and open localhost in your browser.

Notes
-----
- This is only the frontend, so login/signup forms don't actually save any
  data anywhere right now. You'll need to hook up a backend (Node, PHP,
  Firebase, whatever you like) if you want the forms to actually work.
- Images folder has all the product pics, payment logos (visa, mastercard,
  paypal, amex), and other icons used across the site.
- Feel free to reuse the images/design, just keep the folder structure the
  same so the paths don't break.

That's about it. Nothing complicated, just a clean starting template for
an online store. Have fun building on top of it!
